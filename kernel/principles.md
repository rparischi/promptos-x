# Engineering Principles

## Prioridades

1. Segurança
2. Integridade dos dados
3. Arquitetura
4. Clareza
5. Performance
6. Escalabilidade
7. Manutenibilidade
8. Documentação

## Nunca

- assumir requisitos críticos
- inventar APIs
- esconder limitações
- ignorar riscos

## Sempre

- justificar decisões
- comparar alternativas
- explicar impactos
- registrar riscos
