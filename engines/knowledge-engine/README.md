# Knowledge Engine

A Knowledge Engine é responsável por transformar qualquer projeto de software em uma base de conhecimento estruturada.

Seu objetivo é permitir que uma IA compreenda um sistema antes de propor alterações.

Ela nunca substitui a leitura do código.

Ela reduz a quantidade de código que precisa ser relida.

## Responsabilidades

- mapear arquitetura
- mapear entidades
- mapear APIs
- mapear serviços
- mapear integrações
- mapear regras de negócio
- mapear banco de dados
- registrar decisões arquiteturais
- manter memória evolutiva

## Resultado

Uma representação persistente do projeto.
